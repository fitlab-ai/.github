# .github

FitLab AI organization-level configuration and roadmap.

## Structure

```
├── profile/
│   └── README.md          # Organization profile (shown on github.com/fitlab-ai)
├── roadmap/
│   ├── README.md          # Roadmap overview and milestone index
│   ├── phase-1-foundation.md
│   ├── phase-2-integration.md
│   └── phase-3-ecosystem.md
└── README.md              # This file
```

## Usage

- **`profile/README.md`** — 组织首页展示内容，访问 github.com/fitlab-ai 时自动呈现
- **`roadmap/`** — 组织级路线图，描述跨项目的发展阶段和里程碑
