# FitLab AI

**Building the infrastructure for AI-powered software engineering.**

FitLab AI develops tools, frameworks, and collaboration systems that enable AI agents to work together on real-world software engineering tasks.

## Projects

| Project | Description | Status |
|---------|-------------|--------|
| [fit-framework](https://github.com/fitlab-ai/fit-framework) | Enterprise AI development framework (Java) — function compute, workflow orchestration, LLM integration | Stable |
| [agent-infra](https://github.com/fitlab-ai/agent-infra) | AI multi-tool collaboration infrastructure — unified skills & workflows for Claude Code, Codex, Gemini CLI, OpenCode | Active |
| [agora](https://github.com/fitlab-ai/agora) | Multi-AI agent collaboration system — agents simulate human teams with personalities, memory, and communication | Alpha |
